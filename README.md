# Grafos em C++

Projeto desenvolvido durante os estudos de Estruturas de Dados utilizando Grafos direcionados representados por listas de adjacência.

## Funcionalidades

* Criação de grafos direcionados
* Inserção de arestas com peso
* Consulta de vizinhos
* Consulta de pesos
* Cálculo do grau dos vértices
* Verificação de conectividade
* Análise de redes
* Verificação de caminhos Eulerianos

## Conceitos Trabalhados

* Grafos
* Lista de Adjacência
* Teoria dos Grafos
* Redes de Computadores
* Caminhos Eulerianos
* Estruturas de Dados
* Algoritmos

## Estrutura do Projeto

### graph.h / graph.cpp

Implementação da estrutura de grafo contendo:

* Inserção de arestas
* Grau dos vértices
* Lista de vizinhos
* Consulta de peso
* Verificação de conectividade

### network.cpp

Problema de análise de redes.

Objetivo:

* Identificar vértices sobrecarregados comparando capacidade e tráfego da rede.

### domino.cpp

Problema de dominós utilizando grafos.

Objetivo:

* Verificar se existe uma trilha Euleriana capaz de utilizar todas as peças.
